# Image Pack Guide for Slides and Report

This folder contains Mermaid source files for visuals you can use in both your thesis report and presentation.

## Files
- `system_architecture.mmd`: Full system architecture.
- `indexing_pipeline.mmd`: End-to-end indexing process.
- `retrieval_workflow.mmd`: Query workflow with dual-layer routing.
- `evaluation_framework.mmd`: Evaluation dimensions for methodology chapter.
- `results_latency_chart.mmd`: Reported latency range chart.

## Suggested Slide Mapping
- Slide 6: `system_architecture.mmd`
- Slide 7: `indexing_pipeline.mmd`
- Slide 8: `retrieval_workflow.mmd`
- Slide 11: `results_latency_chart.mmd`
- Slide 13 or 14: `evaluation_framework.mmd`

## Suggested Report Mapping
- Chapter 3.3: `system_architecture.mmd`
- Chapter 3.5: `indexing_pipeline.mmd`
- Chapter 3.6: `retrieval_workflow.mmd`
- Chapter 3.8 / 4.x: `evaluation_framework.mmd`
- Chapter 4.3: `results_latency_chart.mmd`

## Export to PNG/SVG (quick options)
1. Open any `.mmd` file.
2. Use a Mermaid preview extension in VS Code.
3. Export as PNG or SVG.

Alternative: paste the `.mmd` content into mermaid.live and export directly.
